Para utilizar el programa correctamente se debe agregar a la misma carpeta donde se encuentra el programa 'pixelart.py' un archivo con el nombre 'codigo.txt'.
Luego para obtener los pixelarts debes simplemente correr el programa (No es necesario cambiar algun valor o agregar algo al codigo, con haber agregado el txt correctamente es suficiente)

* no funciona con instrucciones 'Repetir' anidadas (estas probablemente seran detectadas como error) *
